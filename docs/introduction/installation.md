Bit requires BVM for installation.

## 1. Install Bit Version Manager (BVM)

> [BVM](https://github.com/teambit/bvm) manages Bit installations.

#### Install with NPM

```shell
npm i -g @teambit/bvm
```

#### Install with Yarn

```shell
yarn global add @teambit/bvm
```

## 2. Install Bit (using BVM)

```shell
bvm install
```
